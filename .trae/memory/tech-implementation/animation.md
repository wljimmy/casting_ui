# 动画效果技术记忆

## 实现方式
- **CSS Transitions** - 用于平滑的过渡效果
- **CSS Animations** - 用于复杂的动画效果
- **JavaScript Animations** - 用于需要JS控制的动画

## 过渡效果
- **淡入淡出** - 元素的显示和隐藏
- **滑动** - 元素的滑动进入和退出
- **缩放** - 元素的放大和缩小
- **旋转** - 元素的旋转效果

## 应用场景
- **弹窗** - 弹窗的显示和隐藏动画
- **Toast** - 消息提示的显示和隐藏动画
- **加载动画** - 旋转加载动画
- **图片放大** - 图片的放大和缩小动画
- **导航菜单** - 菜单的展开和收起动画

## 性能考虑
- **使用CSS Transitions** - 优先使用CSS过渡，性能更好
- **避免重排** - 尽量使用transform和opacity属性
- **硬件加速** - 使用transform: translateZ(0)触发硬件加速
- **合理的动画时长** - 动画时长适中，不要过长或过短
