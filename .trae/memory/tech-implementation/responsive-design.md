# 响应式设计技术记忆

## 实现方式
- **Flexbox** - 用于灵活的布局
- **Grid** - 用于网格布局
- **Container Queries** - 用于组件级响应式设计
- **Media Queries** - 适配不同屏幕尺寸

## 适配策略
- **移动优先** - 先设计移动端，再扩展到桌面端
- **弹性单位** - 使用clamp()、vw等弹性单位
- **断点设计** - 针对不同屏幕尺寸设置断点

## 断点设置
- **移动端** - 小于768px
- **平板** - 768px到1024px
- **桌面** - 大于1024px

## 布局调整
- **移动端** - 垂直堆叠布局
- **平板** - 适当的网格布局
- **桌面** - 完整的网格布局

## 组件适配
- **卡片** - 在不同屏幕尺寸下自动调整大小
- **导航** - 在移动端折叠为汉堡菜单
- **表格** - 在移动端变为可滚动或折叠布局
