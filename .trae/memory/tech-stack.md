# 技术栈记忆

## 核心技术
- **HTML5** - 原生HTML5，语义化标签
- **CSS3** - 现代CSS，包括Flexbox、Grid、容器查询
- **JavaScript (ES6+)** - 原生JavaScript，使用ES6+特性

## CSS技术
- **Flexbox** - 用于灵活的布局
- **Grid** - 用于网格布局
- **Container Queries** - 用于组件级响应式设计
- **CSS Variables** - 用于统一管理样式
- **CSS Transitions** - 用于平滑的动画效果
- **Backdrop Filter** - 用于毛玻璃效果

## JavaScript技术
- **ES6+ Features** - 箭头函数、Promise、模板字符串等
- **Fetch API** - 用于动态加载页面
- **DOM Manipulation** - 原生DOM操作
- **Event Handling** - 事件处理
- **Promise** - 用于异步操作

## 性能优化
- **content-visibility: auto** - 优化渲染性能
- **Image Lazy Loading** - 图片懒加载
- **Dynamic Page Loading** - 动态加载页面
- **Minimal Dependencies** - 无外部依赖

## 响应式设计
- **Media Queries** - 适配不同屏幕尺寸
- **Flexible Units** - 使用clamp()、vw等弹性单位
- **Mobile-First** - 移动优先设计

## 浏览器兼容性
- **Modern Browsers** - 支持现代浏览器
- **ES6+ Support** - 支持ES6+特性的浏览器
