# 原生HTML5+CSS+JS UI框架

一套可复用的原生HTML5+CSS+JS UI框架，无框架依赖，可直接浏览器运行，适配手机/Pad/PC多端。

  - 添加字体配置，使用思源宋体作为默认字体

## 快速开始

### 克隆命令
```bash
git clone [仓库地址]
```

### 运行方式
1. 双击 `index.html` 文件即可在浏览器中运行
2. 无需服务器部署，纯静态文件

## 目录结构

```
├── index.html    # 单页展示所有UI控件
├── style.css     # 所有样式定义
├── app.js        # 所有交互逻辑
└── README.md     # 框架文档
```

## 技术特点

- **无依赖**：纯原生HTML5+CSS+JS实现，无需任何框架
- **响应式**：基于Flex+Grid+容器查询实现多端适配
- **现代化**：采用科技蓝(#165DFF)为主的流行配色
- **性能优化**：图片懒加载、content-visibility优化
- **流畅交互**：所有控件带原生过渡动画

## 自定义说明

### 修改配色
在 `style.css` 文件中修改CSS变量定义：

```css
:root {
    /* 主色系 */
    --primary-color: #165DFF;  /* 修改为主色调 */
    /* 其他颜色变量 */
}
```

### 修改尺寸
在 `style.css` 文件中修改尺寸变量：

```css
:root {
    /* 尺寸 */
    --size-xs: 4px;    /* 最小尺寸 */
    --size-sm: 8px;    /* 小尺寸 */
    --size-md: 16px;   /* 中等尺寸 */
    --size-lg: 24px;   /* 大尺寸 */
    --size-xl: 32px;   /* 超大尺寸 */
}
```

### 修改交互
在 `app.js` 文件中修改交互逻辑：

```javascript
// 示例：修改Toast显示逻辑
this.toast.showToast = (type, message, options) => {
    // 自定义实现
};
```
