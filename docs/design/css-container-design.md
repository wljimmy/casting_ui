# CUI 极简框架 · 容器模块设计

## 一、设计总则（不变）

1. 所有框架类统一 `CUI-` 前缀。
2. **页面布局容器仅用 Grid**；Input、菜单、按钮等原子UI不受此限。
3. 容器分：通用容器 + 语义功能容器；布局复用同一套 Grid 规则。
4. 单一职责：视觉 / 内部Grid / 外部占位 严格分离。

***

## 二、视觉容器类（不变）

| 类名        | 核心用途      |
| --------- | --------- |
| CUI-box   | 基础通用容器    |
| CUI-card  | 卡片容器      |
| CUI-fluid | 透明无样式布局容器 |
| CUI-panel | 功能面板容器    |

| CUI-page    | 页面根容器      |

## 三、内部 Grid 布局类（对齐部分**重新拆分独立**）

### 3.1 基础网格

| 类名       | 作用                |
| -------- | ----------------- |
| CUI-grid | 启用 Grid 布局，统一子项间距 |

### 3.2 内部列划分类

| 类名          | 作用           |
| ----------- | ------------ |
| CUI-grid-1c | 内部 1 列（垂直布局） |
| CUI-grid-2c | 内部 2 列等宽     |
| CUI-grid-3c | 内部 3 列等宽     |
| CUI-grid-4c | 内部 4 列等宽     |
| 可扩展至 12c    | 多列均分布局       |

### 3.3 特殊布局类

| 类名              | 作用                                   |
| --------------- | ------------------------------------ |
| CUI-grid-middle | 三列结构：`1fr auto 1fr`，实现左右自适应 + 中间严格居中 |

### 3.4 内部对齐类（**独立方向版，你要的拆分效果**）

统一规则：

- **水平方向**：控制子项在单元格内**左右位置**
- **垂直方向**：控制子项在单元格内**上下位置**
- 每个方向**单独一个类**，可自由组合使用

#### 水平对齐（左右）

| 类名                | 作用           |
| ----------------- | ------------ |
| CUI-grid-left     | 内部子项**水平靠左** |
| CUI-grid-h-center | 内部子项**水平居中** |
| CUI-grid-right    | 内部子项**水平靠右** |

#### 垂直对齐（上下）

| 类名                | 作用           |
| ----------------- | ------------ |
| CUI-grid-top      | 内部子项**垂直靠上** |
| CUI-grid-v-center | 内部子项**垂直居中** |
| CUI-grid-bottom   | 内部子项**垂直靠下** |

#### 组合式完整对齐（保留常用简化类）

| 类名               | 作用              |
| ---------------- | --------------- |
| CUI-grid-center  | 子项**水平+垂直双向居中** |
| CUI-grid-between | 子项两端对齐分布        |
| CUI-grid-around  | 子项间距均匀分布        |
| CUI-grid-stretch | 子项拉伸填满单元格       |

## 四、外部 Grid 占位类（不变）

### 4.1 自身尺寸与对齐

| 类名         | 作用          |
| ---------- | ----------- |
| CUI-full   | 宽高100%占满父容器 |
| CUI-auto   | 宽高由内容自适应    |
| CUI-half   | 宽度50%       |

### 4.2 父网格跨行/跨列

| 类名                          | 作用        |
| --------------------------- | --------- |
| CUI-span-1c \~ CUI-span-12c | 父网格中跨 N 列 |
| CUI-span-1r \~ CUI-span-12r | 父网格中跨 N 行 |

### 4.3 间距类

| 类名           | 作用  |
| ------------ | --- |
| CUI-m-{size} | 外边距 |
| CUI-p-{size} | 内边距 |

## 五、使用示例（对齐独立后的用法）

1. 子项只**水平居中、垂直靠上**
   ```html
   <div class="CUI-card CUI-grid CUI-grid-1c CUI-grid-h-center CUI-grid-top">
     <div>内容</div>
   </div>
   ```
2. 子项**垂直居中、水平靠右**
   ```html
   <div class="CUI-section CUI-grid CUI-grid-1c CUI-grid-right CUI-grid-v-center">
     <div>内容</div>
   </div>
   ```
3. 传统双向居中（直接用简写）
   ```html
   <div class="CUI-box CUI-grid CUI-grid-center">
     <div>内容</div>
   </div>
   ```

***

## 六、规范说明

- 所有对齐类**只作用于容器内部子项**，不影响容器自身占位；
- 水平/垂直类可任意叠加，互不冲突；
- 完全遵循你「布局容器只用 Grid」的规则，不引入 Flex；
- 后续交给AI生成代码时，直接按此表格实现即可。
