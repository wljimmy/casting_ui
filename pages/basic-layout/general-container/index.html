通用容器

通用容器组件提供多种页面内容容器，用于控制内容宽度和布局。

1. 基础容器

基础容器限制最大宽度并居中显示。

.container

最大宽度 1200px，自动居中

参数说明

类名 最大宽度 描述
.container 1200px 基础容器，响应式内边距
.container-fluid 100% 全宽容器

使用说明

<div class="container">
    <!-- 内容区域，最大宽度 1200px -->
</div>

<div class="container-fluid">
    <!-- 全宽内容区域 -->
</div>

是否涉及 JS 动态渲染：否。

2. 固定宽度容器

固定宽度容器提供预设的宽度尺寸。

.container-fixed-sm - 500px
.container-fixed-md - 800px
.container-fixed-lg - 1200px

参数说明

类名 宽度 适用场景
.container-fixed-sm 500px 登录框、小表单
.container-fixed-md 800px 文章详情、中等表单
.container-fixed-lg 1200px 数据展示、复杂页面

使用说明

<!-- 登录框容器 -->
<div class="container-fixed-sm">
    <form>...</form>
</div>

<!-- 文章详情容器 -->
<div class="container-fixed-md">
    <article>...</article>
</div>

是否涉及 JS 动态渲染：否。

3. 网格容器

网格容器提供多列布局方案。

3.1 等分列网格

参数说明

类名 列数 间距
.container-grid-2 2列 24px
.container-grid-3 3列 24px
.container-grid-4 4列 24px
.container-grid-12 12列 24px

使用说明

<!-- 两列布局 -->
<div class="container-grid-2">
    <div>左侧内容</div>
    <div>右侧内容</div>
</div>

<!-- 三列布局 -->
<div class="container-grid-3">
    <div>内容 1</div>
    <div>内容 2</div>
    <div>内容 3</div>
</div>

是否涉及 JS 动态渲染：否。

4. 透明背景容器

透明背景容器用于特殊场景，如深色背景上的内容展示。

透明背景容器

背景色: rgba(255, 255, 255, 0.1)

参数说明

类名 背景色 描述
.transparent-bg-container rgba(255,255,255,0.1) 透明背景容器，毛玻璃效果

使用说明

<div style="background: linear-gradient(...);">
    <div class="transparent-bg-container">
        <!-- 内容 -->
    </div>
</div>

是否涉及 JS 动态渲染：否。

5. 响应式行为

所有容器都支持响应式设计。

容器类型 桌面端 平板端(<1024px) 移动端(<768px)
.container 1200px 100% - 48px 100% - 32px
.container-grid-4 4列 2列 1列
.container-grid-3 3列 2列 1列
.container-grid-2 2列 2列 1列
提示：网格容器在移动端会自动调整为单列布局，确保内容可读性。

是否涉及 JS 动态渲染：否，纯 CSS 媒体查询实现。

6. 容器嵌套

容器可以嵌套使用，实现复杂的布局结构。
