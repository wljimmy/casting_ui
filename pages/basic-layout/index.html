基础布局组件

基础布局组件是构建页面结构的核心元素，包括卡片容器、通用容器等。

1. 卡片容器

卡片容器是一种常用的布局组件，用于展示独立的内容块，具有多种尺寸和样式。

1.1 UI示例

1列卡片

.card.card-size-1

占1列宽度

3列卡片

.card.card-size-3

占3列宽度

6列卡片

.card.card-size-6

占6列宽度

12列卡片

.card.card-size-12

占12列宽度

默认样式

.card.card-style-default

带阴影和边框

毛玻璃样式

.card.card-style-glass

半透明背景

无边框样式

.card.card-style-borderless

无阴影和边框

1.2 参数说明

参数 值 描述
类名 .card 基础卡片类
尺寸 .card-size-1 至 .card-size-12 12列网格系统中的宽度
.card-size-auto 自适应宽度
样式 .card-style-default 默认样式，带阴影和边框
.card-style-glass 毛玻璃效果
.card-style-borderless 无边框样式
圆角 .card-radius-sm 小圆角 (4px)
.card-radius-md 中圆角 (8px)
.card-radius-lg 大圆角 (12px)
.card-radius-none 无圆角
高度 .card-height-auto 高度自适应
默认 固定高度，超出滚动
状态 .card-disabled 禁用状态

1.3 使用说明

适用内容：卡片容器适用于展示独立的内容块，如产品信息、用户资料、新闻文章等。

引用方法：直接在HTML中使用相应的类名。

<!-- 基本卡片 -->
<div class="card card-size-4 card-style-default">
    <h4>卡片标题</h4>
    <p>卡片内容</p>
</div>

<!-- 毛玻璃卡片 -->
<div class="card card-size-4 card-style-glass">
    <h4>毛玻璃卡片</h4>
    <p>半透明背景效果</p>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

2. 通用容器

通用容器用于构建页面的整体布局结构，包括流式容器、固定宽容器、网格容器等。

2.1 UI示例

流式容器

.container-fluid

宽度100%，自适应父容器

小固定宽容器

.container-fixed-sm

宽度500px

网格项1

网格项2

网格项3

2.2 参数说明

参数 值 描述
类名 .container-fluid 流式容器，宽度100%
.container-fixed-sm 小固定宽容器，宽度500px
.container-grid-2 至 .container-grid-12 网格容器，2-12列
.container-height-auto 高度自适应

2.3 使用说明

适用内容：通用容器适用于构建页面的整体布局结构，如页面主体、侧边栏、页脚等。

引用方法：直接在HTML中使用相应的类名。

<!-- 流式容器 -->
<div class="container-fluid">
    <h4>页面主体内容</h4>
    <p>宽度100%，自适应父容器</p>
</div>

<!-- 网格容器 -->
<div class="container-grid-3">
    <div>网格项1</div>
    <div>网格项2</div>
    <div>网格项3</div>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

参数 值 描述

            <li>菜单项2</li>

    </div>

        <h4>内容区域</h4>
        <p>这里是主要内容区域</p>
    </div>
</div>

<!-- 顶部/底部布局 -->
<div>
    <div class="layout-header">
        <h4>顶部导航栏</h4>
    </div>
    <div>
        <h4>内容区域</h4>

    </div>
    <div class="layout-footer">
        <p>底部版权信息</p>
    </div>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

4. 顶部+左小右大布局

顶部+左小右大布局是一种常见的管理后台布局，顶部为导航栏，左侧为菜单栏，右侧为内容区域。

5.1 UI示例

顶部导航栏（高度120px）

左侧菜单栏（宽度200px）

  • 首页
  • 产品管理
  • 用户管理
  • 系统设置

右侧内容区域（占据剩余空间）

这里是主要内容区域，占据除顶部和左侧外的所有空间。

可以根据需要放置各种组件和内容。

5.2 参数说明

参数 值 描述
类名 .layout-top-left-right 顶部+左小右大布局容器
.layout-top-left-right-header 顶部导航栏，高度120px
.layout-top-left-right-sidebar 左侧菜单栏，宽度200px
.layout-top-left-right-content 右侧内容区域，占据剩余空间

5.3 使用说明

适用内容：顶部+左小右大布局适用于管理后台、控制面板等需要清晰导航结构的应用。

引用方法：直接在HTML中使用相应的类名。

<!-- 顶部+左小右大布局 -->
<div class="layout-top-left-right">
    <div class="layout-top-left-right-header">
        <h4>顶部导航栏</h4>
        <!-- 导航栏内容 -->
    </div>
    <div class="layout-top-left-right-sidebar">
        <h4>左侧菜单栏</h4>
        <menu>
            <ul>
                <li>
                    <a href="#">菜单项1</a>
                </li>
                <li>
                    <a href="#">菜单项2</a>
                </li>
            </ul>
        </menu>
    </div>
    <div class="layout-top-left-right-content">
        <h4>右侧内容区域</h4>
        <p>这里是主要内容区域</p>
    </div>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

5. 折叠面板

折叠面板用于展示可展开/收起的内容，节省页面空间。

5.1 UI示例

折叠面板1

▼

折叠面板内容

5.2 参数说明

参数 值 描述
类名 .collapse-panel 折叠面板容器
.collapse-header 折叠面板头部
.collapse-content 折叠面板内容

5.3 使用说明

适用内容：折叠面板适用于展示可展开/收起的内容，如FAQ、详情信息等。

引用方法：直接在HTML中使用相应的类名，需要配合JavaScript实现展开/收起功能。

<!-- 折叠面板 -->
<div class="collapse-panel">
    <div class="collapse-header" onclick="toggleCollapse(this)">
        <h4>折叠面板标题</h4>
        <span>▼</span>
    </div>
    <div class="collapse-content">
        <p>折叠面板内容</p>
    </div>
</div>

<script>
function toggleCollapse(header) {
    const content = header.nextElementSibling;
    const icon = header.querySelector('span');
    
    if (content.style.display === 'none') {
        content.style.display = 'block';
        icon.textContent = '▼';
    } else {
        content.style.display = 'none';
        icon.textContent = '▶';
    }
}
</script>

是否涉及JS动态渲染：是，需要JavaScript实现展开/收起功能。
