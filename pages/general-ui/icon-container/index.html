图标

图标库提供丰富的SVG图标，支持线性和填充两种风格。所有图标均可通过CSS控制大小和颜色。

1. 使用方法

图标使用SVG格式，可直接嵌入HTML或作为背景图使用。

1.1 直接嵌入SVG

<!-- 使用填充风格图标 -->
<svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="currentColor">
  <path d="M12 2l-9 9h3v9h6v-6h4v6h6v-9h3l-9-9z"/>
</svg>

<!-- 使用线性风格图标 -->
<svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
  <path d="M12 2l-9 9h3v9h6v-6h4v6h6v-9h3l-9-9z"/>
</svg>

1.2 通过CSS使用

.icon {
  width: 24px;
  height: 24px;
  display: inline-block;
  background-size: contain;
  background-repeat: no-repeat;
}

是否涉及 JS 动态渲染：否，纯 CSS/SVG。

3. 图标尺寸

图标支持多种尺寸，通过CSS控制。

16px
24px
32px
48px

参数说明

尺寸 用途
16px 小按钮、内联文本
24px 默认尺寸，大多数场景
32px 大按钮、导航项
48px 特征图标、空状态

使用说明

<!-- 小图标 -->
<svg width="16" height="16" viewBox="0 0 24 24" fill="currentColor">...</svg>

<!-- 默认图标 -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="currentColor">...</svg>

<!-- 大图标 -->
<svg width="48" height="48" viewBox="0 0 24 24" fill="currentColor">...</svg>

是否涉及 JS 动态渲染：否。

4. 图标颜色

图标颜色通过CSS的currentColor继承父元素颜色。

主题色
成功色
警告色
错误色

使用说明

<!-- 通过父元素颜色控制 -->
<span style="color: var(--primary-color);">
    <svg width="24" height="24" viewBox="0 0 24 24" fill="currentColor">...</svg>
</span>

<!-- 直接设置颜色 -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="var(--success-color)">...</svg>

是否涉及 JS 动态渲染：否。

5. 图标库结构

图标库按风格分为两种：

风格 路径 特点
填充风格 dev/icons/filled/ 实心图标，适合强调
线性风格 dev/icons/outline/ 描边图标，适合轻量场景
提示：图标库包含1000+个图标，命名采用kebab-case（短横线连接）。完整图标列表请参考图标文件夹。

6. 使用建议

最佳实践：
  • 保持图标尺寸一致，同一界面中尽量使用相同尺寸
  • 使用currentColor让图标继承父元素颜色，便于主题切换
  • 按钮中的图标与文字保持适当间距（8px）
  • 图标应具有明确的语义，避免使用含义模糊的图标
  • 重要操作使用填充风格，次要操作使用线性风格
