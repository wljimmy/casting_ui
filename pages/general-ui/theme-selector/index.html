主题选择器

主题选择器允许用户切换不同的配色方案，即时生效，并且会自动保存到本地存储。

1. UI示例

2. 参数说明

参数 值 描述
方法 openThemeSelector(options) 打开主题选择器
选项 mode 打开模式（modal/inline）
container 内联模式下的容器选择器
方法 ThemeManager 主题管理类
方法 init() 初始化主题管理器
getThemes() 获取所有主题
applyTheme(theme) 应用主题
addTheme(theme) 添加自定义主题

3. 使用说明

适用内容：主题选择器适用于需要支持主题切换的应用，如管理后台、仪表盘等。

引用方法：使用 openThemeSelector() 方法打开主题选择器，或使用 ThemeManager 类管理主题。

<!-- 打开主题选择器 -->
<button class="btn btn-primary" onclick="openThemeSelector({ mode: 'modal' })">打开主题选择器</button>

<!-- 内联主题选择器 -->
<div id="themeContainer"></div>
<button class="btn btn-default" onclick="openThemeSelector({ mode: 'inline', container: '#themeContainer' })">内联打开主题选择器</button>

<script>
// 初始化主题管理器
document.addEventListener('DOMContentLoaded', async function() {
    const themeManager = new ThemeManager();
    await themeManager.init();
    
    // 应用默认主题
    themeManager.applyTheme(themeManager.getThemes()[0]);
    
    // 添加自定义主题
    themeManager.addTheme({
        name: '自定义主题',
        colors: {
            primary: '#FF6B6B',
            secondary: '#4ECDC4',
            success: '#4CAF50',
            warning: '#FFC107',
            error: '#F44336',
            text: {
                primary: '#333333',
                secondary: '#666666'
            },
            background: {
                primary: '#FFFFFF',
                secondary: '#F5F5F5'
            }
        }
    });
});
</script>

是否涉及JS动态渲染：是，通过JavaScript管理主题切换和存储。

4. 功能特点

  • 支持多种主题切换
  • 即时生效，无需刷新页面
  • 自动保存到本地存储
  • 支持自定义主题
  • 提供弹窗和内联两种打开模式
