颜色工具

颜色选择 Input 组件与相关工具

颜色选择输入框

框架提供了自动化的颜色选择 Input 组件，支持原生选择器和自定义颜色选择器两种模式。

1. 原生颜色选择器（默认）

直接使用原生的颜色选择器，无需任何额外配置：

<input type="color" name="nativeColor" value="#165DFF">

说明：这是浏览器原生的颜色选择器，外观和行为因浏览器而异。

2. 自定义颜色选择器

添加 data-cui-color-picker="true" 属性，使用框架自带的颜色选择器：

<input type="color" name="customColor" value="#C2185B" data-cui-color-picker="true">

说明：使用框架的自定义颜色选择器，提供统一的用户体验和更多功能。

3. 在表单中使用

<form id="colorForm">
    <input type="color" name="themeColor" value="#165DFF" data-cui-color-picker="true">
    <input type="color" name="bgColor" value="#FFFFFF">
    <input type="color" name="textColor" value="#333333">
</form>

4. 获取和设置颜色值

使用框架的输入框注册表 API 操作颜色值：

// 获取颜色值
const themeColor = CUI.input.getValue('colorForm.themeColor');
console.log('主题颜色:', themeColor);

// 设置颜色值
CUI.input.setValue('colorForm.themeColor', '#FF5722');

// 获取整个表单的颜色数据
const formData = CUI.input.getFormData('colorForm');
console.log('表单颜色数据:', formData);

5. 属性说明

属性 类型 默认值 说明
type string - 必须为 "color"
name string - 表单字段名称，用于表单数据收集
value string #000000 初始颜色值，必须是有效的 HEX 颜色值
data-cui-color-picker string - 设置为 "true" 使用自定义颜色选择器

6. 两种模式对比

特性 原生选择器 自定义选择器
外观一致性 因浏览器而异 统一风格
功能丰富度 基础功能 完整功能（色盘、预设色、格式转换）
颜色格式 仅 HEX 支持 HEX、RGB、RGBA
预设颜色 无 支持自定义预设
使用方式 直接使用 需添加 data 属性

颜色工具函数

框架提供了一些实用的颜色处理工具函数：

颜色格式转换

// HEX 转 RGB
function hexToRgb(hex) {
    const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
    return result ? {
        r: parseInt(result[1], 16),
        g: parseInt(result[2], 16),
        b: parseInt(result[3], 16)
    } : null;
}

// RGB 转 HEX
function rgbToHex(r, g, b) {
    return '#' + [r, g, b].map(x => {
        const hex = x.toString(16);
        return hex.length === 1 ? '0' + hex : hex;
    }).join('');
}

颜色对比

// 计算颜色对比度
function getContrastRatio(color1, color2) {
    const rgb1 = hexToRgb(color1);
    const rgb2 = hexToRgb(color2);
    
    const lum1 = (0.2126 * rgb1.r + 0.7152 * rgb1.g + 0.0722 * rgb1.b) / 255;
    const lum2 = (0.2126 * rgb2.r + 0.7152 * rgb2.g + 0.0722 * rgb2.b) / 255;
    
    const ratio = (Math.max(lum1, lum2) + 0.05) / (Math.min(lum1, lum2) + 0.05);
    return ratio.toFixed(2);
}

// 判断是否应该使用深色文字
function shouldUseDarkText(backgroundColor) {
    const rgb = hexToRgb(backgroundColor);
    const luminance = (0.2126 * rgb.r + 0.7152 * rgb.g + 0.0722 * rgb.b) / 255;
    return luminance > 0.5;
}
