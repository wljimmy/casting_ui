交互控件

交互控件是用户与页面进行交互的核心元素，包括按钮、输入框、选择控件等。

1. 按钮

按钮是最基本的交互控件，用于触发操作或提交表单。

1.2 参数说明

参数 值 描述
类名 .btn 基础按钮类
尺寸 .btn-sm 小按钮
.btn-md 中按钮（默认）
.btn-lg 大按钮
样式 .btn-primary 主要按钮
.btn-secondary 次要按钮
.btn-text 文本按钮
.btn-icon 图标按钮
状态 .btn-loading 加载状态
disabled 禁用状态

1.3 使用说明

适用内容：按钮适用于触发操作、提交表单、导航等场景。

引用方法：直接在HTML中使用相应的类名。

<!-- 主要按钮 -->
<button class="btn btn-primary">提交</button>

<!-- 图标按钮 -->
<button class="btn btn-icon btn-primary">
    <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <circle cx="12" cy="12" r="10"></circle>
        <line x1="12" y1="8" x2="12" y2="12"></line>
        <line x1="12" y1="16" x2="12.01" y2="16"></line>
    </svg>
</button>

是否涉及JS动态渲染：否，纯CSS组件。

2.2 参数说明

参数 值 描述
类名 .input 单行输入框
.textarea 多行输入框
.input-group 输入框组，包含标签和输入框
.input-label 输入框标签
.input-hint 输入框提示信息
.input-error 输入框错误信息
状态 .error 错误状态
.success 成功状态
disabled 禁用状态

2.3 使用说明

适用内容：输入框适用于用户输入文本、数字、密码等信息的场景。

引用方法：直接在HTML中使用相应的类名。

<!-- 单行输入框 -->
<div class="input-group">
    <label class="input-label">用户名</label>
    <input type="text" class="input" placeholder="请输入用户名">
    <span class="input-hint">请输入您的用户名</span>
</div>

<!-- 多行输入框 -->
<div class="input-group">
    <label class="input-label">备注</label>
    <textarea class="textarea" placeholder="请输入备注信息"></textarea>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

选择控件用于用户从多个选项中选择一个或多个值。

3.2 参数说明

参数 值 描述
类名 .radio-group 单选框组
.radio-item 单选框项
.checkbox-group 复选框组
.checkbox-item 复选框项
.select 下拉选择器
.switch 开关
.switch-sm 小开关
.switch-lg 大开关
.slider 滑块
.date-input 日期选择器

3.3 使用说明

适用内容：选择控件适用于用户从多个选项中选择一个或多个值的场景。

引用方法：直接在HTML中使用相应的类名。

<!-- 单选框 -->
<div class="input-group">
    <label class="input-label">性别</label>
    <div class="radio-group">
        <div class="radio-item">
            <input id="male" type="radio" name="gender">
            <label for="male">男</label>
        </div>
        <div class="radio-item">
            <input id="female" type="radio" name="gender">
            <label for="female">女</label>
        </div>
    </div>
</div>

<!-- 开关 -->
<div class="input-group">
    <label class="input-label">启用通知</label>
    <label class="switch">
        <input type="checkbox">
        <span class="switch-slider"></span>
    </label>
</div>

是否涉及JS动态渲染：否，纯CSS组件。
