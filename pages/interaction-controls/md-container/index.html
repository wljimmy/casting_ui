MD编辑器演示

本演示展示了Casting UI框架的MD富文本输入和渲染模块，包含最新的功能特性。

最新功能特性

  • 重复设置即为取消：如果将p设置为h1，则变为h1；如果h1设置为h1，则删除h1标签，恢复为p标签
  • 默认使用p标签：如果一段文字没有任何标签，则默认使用p标签
  • 自动初始化：通过data-md-editor属性自动初始化编辑器
  • 实时MD解析：支持Markdown语法的实时解析
  • 快捷键支持：Enter换行创建同类标签，Shift+Enter标签内换行

  • 选中文字，点击H1按钮 → 变为H1标签

  • Shift+Enter：标签内换行

高级使用

JavaScript API

除了自动初始化外，也可以通过JavaScript API手动控制编辑器：

// 创建MD编辑器 (无初始内容) const editor = mdInput('container-id'); // 创建MD编辑器 (带初始内容) const editor = mdInput('container-id', { initialValue: '# 初始内容\n\n你好世界!' }); // 获取值 (返回MD格式) const value = editor.getValue(); // 设置值 (使用MD格式) editor.setValue('# 新内容'); // 销毁编辑器 editor.destroy();
MD解析器API

可以单独使用MD解析器功能：

// 解析markdown为HTML const html = mdParse('# 标题\n\n**粗体**文本'); // 解析HTML为markdown const md = htmlToMd('
