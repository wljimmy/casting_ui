卡片容器

卡片容器（CUI-card）用于创建带阴影和圆角的卡片效果，是最常用的视觉容器之一。

基础卡片

默认卡片

这是默认样式的卡片，包含阴影、圆角和内边距。

<div class="CUI-card">
    <h4>默认卡片</h4>
    <p>这是默认样式的卡片，包含阴影、圆角和内边距。</p>
</div>

网格布局类说明

类名 作用
CUI-grid 启用 Grid 布局，统一子项间距
CUI-grid-1c ~ CUI-grid-12c 内部 1-12 列等宽布局
CUI-grid-middle 三列结构：1fr auto 1fr
CUI-grid-left / center / right 水平对齐方式
CUI-grid-top / v-center / bottom 垂直对齐方式

对齐类组合示例
