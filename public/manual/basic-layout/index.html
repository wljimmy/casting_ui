基础布局组件

基础布局组件是构建页面结构的核心元素，包括卡片容器、通用容器等。

1. 卡片容器

卡片容器是一种常用的布局组件，用于展示独立的内容块，具有多种尺寸和样式。

1.2 参数说明

参数 值 描述
类名 .CUI-card 基础卡片类
类名 .CUI-box 基础容器类，带边框
类名 .CUI-fluid 流式容器，宽度100%
类名 .CUI-panel 面板容器，带边框和圆角

1.3 使用说明

适用内容：卡片容器适用于展示独立的内容块，如产品信息、用户资料、新闻文章等。

引用方法：直接在HTML中使用相应的类名。

<!-- 基本卡片 -->
<div class="CUI-card">
    <h4>卡片标题</h4>
    <p>卡片内容</p>
</div>

<!-- 带边框的容器 -->
<div class="CUI-box">
    <h4>容器标题</h4>
    <p>容器内容</p>
</div>

是否涉及JS动态渲染：否，纯CSS组件。

参数 值 描述

.CUI-grid-2c 至 .CUI-grid-4c 网格容器，2-4列

2.3 使用说明

适用内容：通用容器适用于构建页面的整体布局结构，如页面主体、侧边栏、页脚等。

引用方法：直接在HTML中使用相应的类名。

<!-- 流式容器 -->
<div class="CUI-fluid">
    <h4>页面主体内容</h4>
    <p>宽度100%，自适应父容器</p>
</div>

<div class="CUI-grid CUI-grid-3c">
    <div>网格项1</div>

    <div>网格项3</div>
</div>

3. 布局容器

布局容器用于构建复杂的页面布局，支持两种布局模式：网页布局和应用壳布局。

3.1 布局模式

Casting UI 提供两种布局模式，通过在 body 元素上添加相应的类来切换：

布局模式 类名 描述
网页布局 layout-web 流式自适应，超宽屏时主内容区限宽居中
应用壳布局 layout-shell 上下左右固定，中间主内容区独立滚动

3.2 HTML 结构

布局使用固定的 5 个原生区块结构：

<body class="layout-web">
  <header>顶部</header>
  <aside class="sidebar-left">左侧边栏</aside>
  <main>主内容区</main>
  <aside class="sidebar-right">右侧边栏</aside>
  <footer>底部</footer>
</body>

3.3 控制类

通过在 body 元素上添加以下类来控制布局元素的显示和行为：

控制类 描述 适用布局
hidden-header 隐藏头部 两种布局
hidden-sidebar-left 隐藏左侧边栏 两种布局
hidden-sidebar-right 隐藏右侧边栏 两种布局
hidden-footer 隐藏底部 两种布局
sticky-sidebar 边栏吸顶 仅网页布局

3.4 使用说明

适用内容：布局容器适用于构建复杂的页面布局，如管理后台、博客、应用界面等。

引用方法：在 body 元素上添加相应的布局类和控制类。

<!-- 网页布局示例 -->
<body class="layout-web">
  <header>顶部导航</header>
  <aside class="sidebar-left">左侧菜单</aside>
  <main>主内容</main>
  <aside class="sidebar-right">右侧信息</aside>
  <footer>底部版权</footer>
</body>

<!-- 应用壳布局示例（带边栏吸顶） -->
<body class="layout-shell">
  <header>顶部导航</header>
  <aside class="sidebar-left">左侧菜单</aside>
  <main>主内容</main>
  <aside class="sidebar-right">右侧信息</aside>
  <footer>底部版权</footer>
</body>

3.5 布局示例

查看 布局示例页面 了解更多布局功能。

是否涉及JS动态渲染：否，纯CSS组件，但可以通过JavaScript动态添加/移除控制类来实现交互效果。

4. 布局应用示例

使用新的布局系统，可以轻松实现各种常见的布局模式：

4.1 管理后台布局

使用应用壳布局，实现固定的顶部导航栏、左侧菜单栏和底部信息栏，中间为可滚动的主内容区：

<body class="layout-shell">
  <header>顶部导航栏</header>
  <aside class="sidebar-left">左侧菜单栏</aside>
  <main>主内容区</main>
  <aside class="sidebar-right">右侧信息栏</aside>
  <footer>底部版权信息</footer>
</body>

4.2 博客布局

使用网页布局，实现响应式的博客页面，超宽屏时主内容区居中显示：

<body class="layout-web">
  <header>博客标题和导航</header>
  <aside class="sidebar-left">分类和标签</aside>
  <main>博客文章内容</main>
  <aside class="sidebar-right">作者信息和推荐</aside>
  <footer>底部链接和版权</footer>
</body>

4.3 单栏布局

通过隐藏侧边栏，实现单栏布局：

<body class="layout-web hidden-sidebar-left hidden-sidebar-right">
  <header>网站标题</header>
  <aside class="sidebar-left">左侧边栏</aside>
  <main>主内容区</main>
  <aside class="sidebar-right">右侧边栏</aside>
  <footer>底部信息</footer>
</body>

5. 折叠面板

折叠面板用于展示可展开/收起的内容，节省页面空间。

5.1 UI示例

5.2 参数说明

参数 值 描述
类名 .CUI-box 折叠面板容器
类名 .collapse-header 折叠面板头部
类名 .collapse-content 折叠面板内容

5.3 使用说明

适用内容：折叠面板适用于展示可展开/收起的内容，如FAQ、详情信息等。

引用方法：直接在HTML中使用相应的类名，需要配合JavaScript实现展开/收起功能。

<!-- 折叠面板 -->
<div class="CUI-box">
    <div class="collapse-header" onclick="toggleCollapse(this)">
        <h4>折叠面板标题</h4>
        <span>▼</span>
    </div>
    <div class="collapse-content">
        <p>折叠面板内容</p>
    </div>
</div>

<script>
function toggleCollapse(header) {
    const content = header.nextElementSibling;
    const icon = header.querySelector('span');
    
    if (content.style.display === 'none') {
        content.style.display = 'block';
        icon.textContent = '▼';
    } else {
        content.style.display = 'none';
        icon.textContent = '▶';
    }
}
</script>

是否涉及JS动态渲染：是，需要JavaScript实现展开/收起功能。
