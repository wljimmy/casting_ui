布局容器

布局容器专注于页面级布局结构，配合 Grid 系统实现灵活的页面排版。

基础布局结构

CUI-header

页面头部

CUI-main

主内容区域

CUI-footer

页面底部

<body class="CUI-page">
    <header class="CUI-header">
        <h1>页面头部</h1>
    </header>
    <main class="CUI-main">
        <p>主内容区域</p>
    </main>
    <footer class="CUI-footer">
        <p>页面底部</p>
    </footer>
</body>

带侧边栏的布局

CUI-aside

侧边栏

CUI-main

主内容区域

<body class="CUI-page">
    <aside class="CUI-aside">
        <menu>...</menu>
    </aside>
    <main class="CUI-main">
        <p>主内容区域</p>
    </main>
</body>

约束宽度布局

CUI-wrap

内容宽度约束包裹容器，最大宽度 1200px，居中显示。

<div class="CUI-wrap">
    <h1>页面标题</h1>
    <main class="CUI-main">
        <p>主内容区域，最大宽度被约束</p>
    </main>
</div>

Grid 布局类完整说明

分类 类名 作用
基础网格 CUI-grid 启用 Grid 布局，统一子项间距
CUI-grid-middle 三列结构：1fr auto 1fr
内部列划分 CUI-grid-1c 内部 1 列（垂直布局）
CUI-grid-2c 内部 2 列等宽
CUI-grid-3c 内部 3 列等宽
CUI-grid-4c 内部 4 列等宽
... ~ CUI-grid-12c 可扩展至 12 列
CUI-grid-center 水平+垂直双向居中
水平对齐 CUI-grid-left 内部子项水平靠左
CUI-grid-h-center 内部子项水平居中
CUI-grid-right 内部子项水平靠右
垂直对齐 CUI-grid-top 内部子项垂直靠上
CUI-grid-v-center 内部子项垂直居中
CUI-grid-bottom 内部子项垂直靠下
