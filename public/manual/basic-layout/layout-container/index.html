布局容器

布局容器组件用于构建页面的整体结构，提供多种常见的页面布局方案。

1. 基础布局容器

基础布局容器采用侧边栏+内容区的经典布局。

侧边栏

固定宽度 200px

内容区

自适应剩余空间

参数说明

类名 描述
.layout-container 布局容器，使用 Grid 布局
.layout-sidebar 侧边栏区域，固定宽度 200px
.layout-content 内容区域，自适应剩余空间

使用说明

适用场景：管理后台、文档站点等需要侧边栏导航的页面。

引用方法：纯 CSS 组件，无需 JavaScript。

<div class="layout-container">
    <div class="layout-sidebar">
        <!-- 侧边栏内容 -->
    </div>
    <div class="layout-content">
        <!-- 主内容区 -->
    </div>
</div>

是否涉及 JS 动态渲染：否。

2. 顶部+左小右大布局

顶部导航+左侧边栏+右侧内容的经典管理后台布局。

顶部 Header 固定高度 120px
侧边栏

固定宽度 200px

内容区

自适应剩余空间

参数说明

类名 描述
.layout-top-left-right 布局容器，最小高度 100vh
.layout-top-left-right-header 顶部区域，固定高度 120px
.layout-top-left-right-sidebar 侧边栏区域，固定宽度 200px
.layout-top-left-right-content 内容区域，自适应剩余空间

使用说明

适用场景：管理后台、仪表盘等复杂页面。

引用方法：纯 CSS 组件，无需 JavaScript。

<div class="layout-top-left-right">
    <div class="layout-top-left-right-header">
        <!-- 顶部内容 -->
    </div>
    <div class="layout-top-left-right-sidebar">
        <!-- 侧边栏内容 -->
    </div>
    <div class="layout-top-left-right-content">
        <!-- 主内容区 -->
    </div>
</div>

是否涉及 JS 动态渲染：否。

3. 完整布局（含底部）

包含头部、侧边栏、内容区和底部的完整布局。

侧边栏
Header
内容区
Footer

参数说明

类名 描述
.layout-header 头部区域，最小高度 60px
.layout-footer 底部区域，最小高度 60px

使用说明

适用场景：需要完整页面结构的场景。

<div class="layout-container">
    <div class="layout-sidebar">...</div>
    <div class="layout-header">...</div>
    <div class="layout-content">...</div>
    <div class="layout-footer">...</div>
</div>

是否涉及 JS 动态渲染：否。

4. Header 元素位置控制

用于控制 Header 内元素的对齐方式。

左侧元素 右对齐元素

参数说明

类名 描述
.header-element-right 将元素靠右对齐
.header-element-center 将元素居中对齐

使用说明

<header class="layout-top-left-right-header">
    <div>Logo</div>
    <div class="header-element-right">
        <!-- 右侧内容，如搜索框、用户头像等 -->
    </div>
</header>

是否涉及 JS 动态渲染：否。

5. Header 搜索框

优雅的头部搜索框样式。

参数说明

6. 响应式设计

所有布局容器都支持响应式设计，在移动端会自动调整为单列布局。

提示：在屏幕宽度小于 768px 时，布局容器会自动切换为单列布局，侧边栏会移至顶部。

响应式行为

布局类型 桌面端 移动端(<768px)
基础布局 侧边栏+内容区 单列，侧边栏在上
顶部+左小右大 Header+侧边栏+内容区 单列堆叠

是否涉及 JS 动态渲染：否，纯 CSS 媒体查询实现。
