颜色选择器

颜色选择器是 Casting UI 框架的核心模块，提供直观的颜色选择界面，支持多种颜色格式转换。

功能特性

  • 支持预设颜色快速选择
  • 标准色盘自由选取
  • 亮度调节滑块
  • 手动输入颜色值
  • 颜色格式转换（HEX/RGB/RGBA）
  • 实时颜色预览

快速开始

import { openColorPicker } from '/modules/js/color-picker.js';

// 基础用法
const color = await openColorPicker();
console.log('选择的颜色:', color);

// 带选项的用法
const result = await openColorPicker({
    presetColors: ['#165DFF', '#67C23A', '#E6A23C'],
    format: 'hex',
    initialColor: '#FF5733'
});

API 选项

选项 类型 默认值 说明
presetColors Array 内置15种颜色 预设颜色数组
format String hex 输出格式：hex/rgb/rgba
initialColor String #165DFF 初始颜色

颜色格式

HEX

#FF5733

#165DFF

#FFFFFF

RGB

rgb(255, 87, 51)

rgb(22, 93, 255)

rgb(255, 255, 255)

RGBA

rgba(255, 87, 51, 1)

rgba(22, 93, 255, 0.5)

rgba(255, 255, 255, 0.8)
