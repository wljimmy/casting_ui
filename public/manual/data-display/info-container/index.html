进度条控制接口

使用 dom.init(container, options) 初始化进度条，返回控制接口。

方法 说明 示例
dom.init() 初始化进度条 dom.init('#my-progress', { showLabel: true })
.go(value) 设置进度值（0-100） myProgress.go(50)
.increment(delta) 增加进度 myProgress.increment(10)
.decrement(delta) 减少进度 myProgress.decrement(5)
.reset() 重置到0 myProgress.reset()
.complete() 完成到100 myProgress.complete()
.getValue() 获取当前值 myProgress.getValue()
.get(id) 获取已存在的进度条 dom.get('#my-progress')
