状态栏

状态栏组件用于显示系统状态、通知和指示信息。支持多种样式变体、闪动效果和专属图标。采用纯CSS实现，通过mask技术实现SVG图标显示和颜色自定义。

属性速查表

属性 说明
data-type 状态类型：info, success, warning, error, wifi, cpu, memory等
data-flash 闪动效果：true/false
data-symbol 符号模式：emoji（强制使用颜文字符号）
title 鼠标悬停提示文本
style="--status-color" 自定义颜色
