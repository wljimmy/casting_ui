反馈组件

反馈组件用于向用户提供操作反馈，包括弹窗、Toast、加载动画等。

1. 弹窗

弹窗用于显示重要信息或请求用户操作。

1.1 UI示例

1.2 参数说明

参数 值 描述
方法 showModal(options) 显示弹窗
选项 title 弹窗标题
content 弹窗内容
buttons 按钮配置数组
position 弹窗位置（center/top/bottom/left/right）
glass 是否使用毛玻璃效果
closeOnOverlayClick 点击遮罩是否关闭弹窗

1.3 使用说明

适用内容：弹窗适用于显示重要信息、请求用户确认、收集用户输入等场景。

引用方法：使用 showModal() 方法创建弹窗。

<!-- 触发按钮 -->
<button class="btn btn-primary" onclick="showModalExample()">显示弹窗</button>

<script>
function showModalExample() {
    showModal({
        title: '示例弹窗',
        content: '这是一个使用 Casting UI 框架创建的弹窗',
        buttons: [
            {
                text: '确定',
                style: 'primary',
                action: () => {
                    showToast('您点击了确定', 'success');
                }
            },
            {
                text: '取消',
                action: () => {
                    showToast('您点击了取消', 'info');
                }
            }
        ],
        position: 'center',
        glass: false
    });
}
</script>

是否涉及JS动态渲染：是，通过JavaScript创建和管理弹窗。

2. Toast

Toast用于显示临时的消息提示。

2.1 UI示例

2.2 参数说明

参数 值 描述
方法 showToast(message, type, position) 显示Toast消息
参数 message 消息内容
type 消息类型（success/error/warning/info）
position 消息位置（top/bottom，默认top）

2.3 使用说明

适用内容：Toast适用于显示操作结果、系统通知等临时消息。

引用方法：使用 showToast() 方法显示Toast消息。

<!-- 触发按钮 -->
<button class="btn btn-primary" onclick="showToastExample()">显示Toast</button>

<script>
function showToastExample() {
    // 显示成功消息
    showToast('操作成功', 'success');
    
    // 显示错误消息
    // showToast('操作失败', 'error');
    
    // 显示警告消息
    // showToast('警告信息', 'warning');
    
    // 显示信息消息
    // showToast('提示信息', 'info');
    
    // 显示底部Toast
    // showToast('底部消息', 'success', 'bottom');
}
</script>

是否涉及JS动态渲染：是，通过JavaScript创建和管理Toast。

3. 加载类

加载类组件用于显示加载状态，包括加载动画和骨架屏。

3.1 UI示例

卡片骨架屏
列表骨架屏
表格骨架屏

3.2 参数说明

参数 值 描述
类名 .loading 基础加载动画
.loading-sm 小尺寸加载动画
.loading-lg 大尺寸加载动画
.skeleton 骨架屏基础类
.skeleton-card 卡片骨架屏
.skeleton-list 列表骨架屏
.skeleton-table 表格骨架屏
方法 showLoading() 显示加载遮罩
hideLoading() 隐藏加载遮罩

3.3 使用说明

适用内容：加载类组件适用于显示数据加载、页面加载等状态。

引用方法：直接在HTML中使用骨架屏类，或使用 showLoading() 方法显示加载遮罩。

<!-- 加载动画 -->
<div class="loading"></div>

<!-- 卡片骨架屏 -->
<div class="skeleton skeleton-card"></div>

<!-- 加载遮罩 -->
<button class="btn btn-primary" onclick="showLoadingExample()">显示加载遮罩</button>

<script>
function showLoadingExample() {
    // 显示加载遮罩
    showLoading();
    
    // 模拟加载完成
    setTimeout(() => {
        hideLoading();
        showToast('加载完成', 'success');
    }, 2000);
}
</script>

是否涉及JS动态渲染：骨架屏为纯CSS组件，加载遮罩通过JavaScript管理。

4. 信息提示

信息提示用于向用户展示不同类型的提示信息，包括信息、成功、警告和错误等状态。

4.1 UI示例

这是一个基础信息提示，用于展示一般性的提示内容。
操作成功！您的更改已保存。
请注意：此操作不可逆，请谨慎处理。
操作失败！请检查网络连接后重试。

系统更新

系统将于今晚进行维护更新，预计耗时2小时。

4.2 参数说明

参数 值 描述
类名 .message 基础信息提示类
类型 .message-info 信息类型（蓝色）
.message-success 成功类型（绿色）
.message-warning 警告类型（黄色）
.message-error 错误类型（红色）

4.3 使用说明

适用内容：信息提示适用于向用户展示状态反馈、操作结果、系统通知等场景。

引用方法：直接在HTML中使用相应的类名。

<!-- 基础信息提示 -->
<div class="message message-info">
    <div>这是一个基础信息提示</div>
</div>

<!-- 带标题的信息提示 -->
<div class="message message-success">
    <h4>操作成功</h4>
    <div>您的更改已保存</div>
</div>

是否涉及JS动态渲染：否，纯CSS组件。
