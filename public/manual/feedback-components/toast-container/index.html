Toast

Toast 是一种轻量级的提示组件，用于在屏幕特定位置显示短暂的消息通知。

基础 Toast

点击下方按钮查看 Toast 效果：

<button class="CUI-btn CUI-btn-primary" onclick="showToast('success', '操作成功')">成功Toast</button>

showToast(type, message, position)

// type: 'success' | 'error' | 'warning' | 'info'
// message: 提示内容
// position: 'top' | 'center' | 'bottom' (可选，默认 'top')

不同位置的 Toast

JS API

参数 类型 说明
type string Toast 类型：success / error / warning / info
message string Toast 显示的消息内容
position string 显示位置：top / center / bottom（可选，默认 top）

使用示例

// 成功提示
showToast('success', '保存成功！');

// 错误提示
showToast('error', '保存失败，请重试');

// 警告提示
showToast('warning', '存储空间不足');

// 信息提示
showToast('info', '有新版本可用');

// 底部显示
showToast('success', '操作完成', 'bottom');
