身份证高级验证器（IDCard Validator）

身份证高级验证器提供完整的身份证号码解析和验证功能，支持分段解析（省/市/区/生日/性别）、校验码验证（ISO 7064 MOD 11-2）、实时解析和历史行政区划数据支持。

快速上手

只需为输入框添加 class="CUI-input-box" 和 data-validate="idcard-adv"，框架自动完成身份证解析和验证：

<input type="text" name="idcard" class="CUI-input-box" 
       data-label="身份证号码" 
       data-validate="idcard-adv" 
       placeholder="请输入18位身份证号">

功能特性

分段解析

自动解析省、市、区、出生日期、性别信息

校验码验证

基于 ISO 7064 MOD 11-2 算法验证校验码

实时解析

输入时逐位解析，即时显示解析结果

历史数据支持

支持已撤销行政区划（如莱芜市、汉中地区等）

状态挂靠

解析状态挂靠到注册表，可通过 API 获取详细信息

自动提示

自动设置默认提示信息和错误信息

验证规则对比

验证类型 触发条件 功能描述
data-validate="idcard" 普通验证 仅正则匹配（18位数字或17位数字+X）
data-validate="idcard-adv" 高级验证 完整解析（省/市/区/生日/性别）+ 校验码验证

解析状态输出

解析状态挂靠在注册表节点的 idcard 属性中，包含以下字段：

字段类型说明
statusstring状态：waiting/parsing/valid/error
phasenumber解析阶段（0-6）
provinceobject省级信息：{code, name, valid}
cityobject地级信息：{code, name, valid}
districtobject县级信息：{code, name, valid}
birthobject出生日期：{date, year, month, day, age, valid}
genderstring性别：male/female
checksumValidboolean校验码是否有效
isValidboolean整体是否有效
displayTextstring可读的解析结果文本
errorstring错误信息（验证失败时）

获取解析状态示例

// 获取身份证解析状态
const idcardState = CUI.input.get('idcard').idcard;

// 输出解析结果
console.log(idcardState.province.name);  // 如：山东省
console.log(idcardState.city.name);      // 如：济南市
console.log(idcardState.district.name);  // 如：历下区
console.log(idcardState.birth.date);     // 如：1990-01-15
console.log(idcardState.birth.age);      // 年龄
console.log(idcardState.gender);         // male 或 female
console.log(idcardState.isValid);        // 是否有效

验证示例

基础验证

输入身份证号后，失焦时自动验证并显示解析结果

自定义提示信息

可以通过 data-info 和 data-error 自定义提示和错误信息：

<input type="text" name="idcard" class="CUI-input-box" 
       data-label="身份证号" 
       data-validate="idcard-adv" 
       data-info="自定义提示" 
       data-error="自定义错误信息">

表单集成示例

function checkForm() {
    if (CUI.input.isFormValid('idcardForm')) {
        const data = CUI.input.getFormData('idcardForm');
        const idcardState = CUI.input.get('idcardForm.idcard').idcard;
        console.log('表单数据:', data);
        console.log('身份证解析:', idcardState);
        showToast('success', '验证通过！');
    } else {
        showToast('error', '请检查表单');
    }
}

错误类型

错误信息触发条件
身份证号位数不正确输入超过18位
省级代码无效前2位省份代码不存在
区级代码无效前6位行政区划代码不存在
出生日期无效第7-14位不是有效日期
校验码错误第18位校验码不正确
身份证号不完整，请输入18位身份证号失焦时输入位数不足18位

JavaScript API

模块访问

// 访问身份证验证器模块
const validator = window.CUI.idcardValidator;

// 手动解析身份证号
validator.parse('11010119900307723X').then(state => {
    console.log(state);
    // { status: 'valid', province: {...}, city: {...}, ... }
});

parse 方法

参数类型说明
valuestring身份证号码（支持带空格格式）

返回 Promise，resolve 时返回解析状态对象。

validateChecksum 方法

// 单独验证校验码
const isValid = validator.validateChecksum('11010119900307723X');
console.log(isValid);  // true 或 false

与格式化模块配合

格式化效果：11 01 01 1990 03 07 723 X
