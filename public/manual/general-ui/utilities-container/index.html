工具类组件

工具类组件提供常用的辅助样式，用于控制间距、文本样式等。

1. 间距占位符

间距占位符用于在元素之间创建固定间距。

元素 1
.spacer-xs (4px)
元素 2
.spacer-sm (8px)
元素 3
.spacer-md (16px)
元素 4

参数说明

类名 高度 描述
.spacer-xs 4px 超小间距
.spacer-sm 8px 小间距
.spacer-md 16px 中间距
.spacer-lg 24px 大间距
.spacer-xl 32px 超大间距

使用说明

<div>元素 1</div>
<div class="spacer-md"></div>
<div>元素 2</div>

是否涉及 JS 动态渲染：否。

2. 文本样式

文本样式类用于快速设置文本的外观。

2.1 文本颜色

.text-muted - 灰色文本（次要信息）

默认文本 - 主要文本颜色

次要文本 - 描述性文本

2.2 字体样式

.text-serif - 宋体文本（中文衬线字体）

默认无衬线字体

参数说明

类名 样式 描述
.text-muted color: var(--text-secondary) 灰色文本，用于次要信息
.text-serif font-family: 宋体 中文衬线字体

使用说明

<!-- 灰色文本 -->
<p class="text-muted">这是次要信息</p>

<!-- 宋体文本 -->
<p class="text-serif">这是宋体文本</p>

是否涉及 JS 动态渲染：否。

3. 占位符

占位符用于显示加载状态或预留内容位置。

参数说明

类名 样式 描述
.placeholder 背景动画效果 占位符，带闪烁动画效果

使用说明

<!-- 文本占位符 -->
<div class="placeholder" style="width: 100%; height: 20px;"></div>

<!-- 图片占位符 -->
<div class="placeholder" style="width: 200px; height: 150px;"></div>

是否涉及 JS 动态渲染：否。

4. 折叠面板

折叠面板用于展示可展开/收起的内容。

点击展开/收起 ▼

这是折叠面板的内容区域，可以放置任意内容。

参数说明

类名 描述
.collapse-panel 折叠面板容器
.collapse-panel-header 面板头部，点击触发折叠
.collapse-panel-content 面板内容区域

使用说明

<div class="collapse-panel">
    <div class="collapse-panel-header">
        标题
    </div>
    <div class="collapse-panel-content">
        内容区域
    </div>
</div>

是否涉及 JS 动态渲染：是，需要 JavaScript 控制展开/收起状态。

5. 使用建议

最佳实践：
  • 使用间距占位符替代空的 div 标签
  • 文本样式类应与语义化标签配合使用
  • 占位符适合加载状态，骨架屏适合复杂布局
  • 折叠面板内容较多时考虑使用
