初始化指南

欢迎使用 Casting UI 框架！这是一个极简UI组件库，让用户专注内容，做到除了业务内容，其他内容都由框架自动处理。

设计理念

  • 用户向使用 md 一样使用 Casting 框架，无需学习新的语法或工具
  • 仅用 HTML 基础标签划定容器、搭建原生层级
  • 扩展配置仅用 data-* 属性
  • 所有复杂样式、组件DOM结构、事件监听、交互逻辑，全部收敛至框架 JS 与 CSS
  • 用户 HTML 保持极简

基础引用

方式一：使用合并文件（推荐）

<!-- 引入 CSS 文件 -->
<link rel="stylesheet" href="dist/v0.3.0/Casting.css">

<!-- 引入 JavaScript 文件 -->
<script src="dist/v0.3.0/Casting.js"></script>

方式二：使用模块化文件

<!-- 引入 CSS 文件 -->
<link rel="stylesheet" href="src/modules/css/core.css">
<link rel="stylesheet" href="src/modules/css/container.css">
<link rel="stylesheet" href="src/modules/css/layout.css">
<link rel="stylesheet" href="src/modules/css/components.css">

<!-- 引入 JavaScript 文件 -->
<script type="module" src="src/modules/js/index.js"></script>

基本使用

1. 定义基础容器

<menu>
    <ul>
        <li>菜单项 1</li>
        <li>菜单项 2</li>
    </ul>
</menu>

2. 使用容器组件

<div class="CUI-box">
    <!-- 内容 -->
</div>

<div class="CUI-card">
    <!-- 内容 -->
</div>

3. Grid 布局

<div class="CUI-grid CUI-grid-2c">
    <div>第一列</div>
    <div>第二列</div>
</div>

框架架构

模块 文件 功能描述
core core.js 核心模块，提供调试、遮罩、弹窗基类
modal modal.js 弹窗模块，提供模态框功能
message message.js 消息提示模块
menu menu.js 菜单模块，支持多级嵌套
theme-manager theme-manager.js 主题管理模块
image-zoom image-zoom.js 图片放大模块
color-picker color-picker.js 颜色选择器模块
