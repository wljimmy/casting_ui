主题管理器

主题管理器是 Casting UI 框架的核心模块，提供完整的主题切换和自定义主题功能。支持预设主题加载、自定义主题创建、实时预览等特性。

功能特性

  • 支持13种预设主题，覆盖多种设计风格
  • 自定义主题创建、编辑、删除
  • 实时主题预览功能
  • 主题持久化存储
  • 事件监听机制

快速开始

import { ThemeManager } from './modules/js/theme-manager.js';

const themeManager = new ThemeManager();
await themeManager.init();

// 打开主题选择器
try {
    const theme = await themeManager.openThemeSelector();
    console.log('已切换到:', theme.name);
} catch (error) {
    console.log('用户取消');
}

API 方法

方法名 功能描述
init() 初始化主题管理器，加载预设主题和自定义主题
applyTheme(theme) 应用指定主题到页面
addTheme(theme) 添加自定义主题
editTheme(index, theme) 编辑指定索引的自定义主题
deleteTheme(index) 删除指定索引的自定义主题
openThemeSelector(options) 打开主题选择器弹窗
getCurrentTheme() 获取当前主题
getThemes() 获取所有主题列表
on(event, callback) 注册事件监听器

事件监听

// 注册主题切换事件
themeManager.on('themeChanged', (theme) => {
    console.log('主题已切换:', theme.name);
});

// 注册主题添加事件
themeManager.on('themeAdded', (theme) => {
    console.log('主题已添加:', theme.name);
});

// 注册主题更新事件
themeManager.on('themeUpdated', (theme) => {
    console.log('主题已更新:', theme.name);
});

// 注册主题删除事件
themeManager.on('themeDeleted', (theme) => {
    console.log('主题已删除:', theme.name);
});
