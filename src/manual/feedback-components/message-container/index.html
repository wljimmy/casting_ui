信息提示

信息提示组件用于向用户展示不同类型的提示信息，包括信息、成功、警告和错误等状态。

1. 基础信息提示

基础信息提示用于展示一般性的提示信息。

这是一个基础信息提示，用于展示一般性的提示内容。

2. 成功信息提示

成功信息提示用于展示操作成功的状态。

操作成功！您的更改已保存。

3. 警告信息提示

警告信息提示用于展示需要注意的信息。

请注意：此操作不可逆，请谨慎处理。

4. 错误信息提示

错误信息提示用于展示操作失败或错误信息。

操作失败！请检查网络连接后重试。

5. 带标题的信息提示

信息提示可以包含标题，用于更清晰地传达信息。

系统更新

系统将于今晚进行维护更新，预计耗时2小时，期间服务可能不稳定。

提交成功

您的表单已成功提交，我们将在3个工作日内处理您的申请。

存储空间不足

您的存储空间即将用完，请及时清理或升级存储方案。

连接超时

无法连接到服务器，请检查您的网络设置或稍后重试。

6. 带列表的信息提示

信息提示可以包含列表，用于展示多项内容。

系统要求

请确保您的系统满足以下要求：

  • 操作系统：Windows 10 或 macOS 10.15+
  • 内存：至少 4GB RAM
  • 存储空间：至少 2GB 可用空间
  • 浏览器：Chrome 80+, Firefox 75+, Safari 13+

7. 参数说明

参数 值 描述
类名 .message 基础信息提示类
类型 .message-info 信息类型（蓝色）
.message-success 成功类型（绿色）
.message-warning 警告类型（黄色）
.message-error 错误类型（红色）
标题 h4 信息提示标题
内容 div 信息提示内容容器

8. 使用说明

适用内容：信息提示适用于向用户展示状态反馈、操作结果、系统通知等场景。

引用方法：直接在HTML中使用相应的类名。

<!-- 基础信息提示 -->
<div class="message message-info">
    <div>这是一个基础信息提示</div>
</div>

<!-- 带标题的信息提示 -->
<div class="message message-success">
    <h4>操作成功</h4>
    <div>您的更改已保存</div>
</div>

<!-- 带列表的信息提示 -->
<div class="message message-warning">
    <h4>注意事项</h4>
    <div>
        <p>请注意以下事项：</p>
        <ul>
            <li>事项一</li>
            <li>事项二</li>
        </ul>
    </div>
</div>

是否涉及JS动态渲染：否，纯CSS组件。
