初始化指南

欢迎使用 Casting UI 框架！本指南将帮助您快速开始使用我们的极简UI组件库。

1. 基础引用

要使用 Casting UI 框架，您需要在 HTML 页面中引入以下文件。我们提供了两种引用方式：

提示：生产环境推荐使用合并后的文件，可以减少HTTP请求，提高加载速度。

方式一：使用合并文件（推荐）

<!-- 引入 CSS 文件 -->
<link rel="stylesheet" href="dist/v0.2.0/Casting.css">

<!-- 引入 JavaScript 文件 -->
<script src="dist/v0.2.0/Casting.js"></script>

方式二：使用模块化文件

<!-- 引入 CSS 文件 -->
<link rel="stylesheet" href="dev/modules/css/core.css">
<link rel="stylesheet" href="dev/modules/css/components.css">

<!-- 引入 JavaScript 文件 -->
<script src="dev/modules/js/core.js"></script>
<script src="dev/modules/js/modal.js"></script>
<script src="dev/modules/js/message.js"></script>
<script src="dev/modules/js/image-zoom.js"></script>
<script src="dev/modules/js/theme-manager.js"></script>
<script src="dev/modules/js/color-picker.js"></script>
<script src="dev/modules/js/ui.js"></script>
<script src="dev/modules/js/md.js"></script>

2. 框架初始化

在引入框架文件后，您可以通过以下方式初始化框架：

// 初始化 UI 框架
const ui = new UI();

// 初始化主题管理器
const themeManager = new ThemeManager();
await themeManager.init();

// 应用默认主题
themeManager.applyTheme(themeManager.getThemes()[0]);
成功：如果您使用的是合并后的 Casting.js 文件，所有模块会自动初始化，无需手动调用。

3. 框架架构

Casting UI 框架采用模块化设计，主要包含以下模块：

模块 文件 功能描述
core core.js 核心模块，提供调试、遮罩、弹窗基类
modal modal.js 弹窗模块，提供模态框功能
message message.js 消息提示模块，提供Toast、Loading等功能
image-zoom image-zoom.js 图片放大模块，支持点击图片放大查看
theme-manager theme-manager.js 主题管理模块，支持主题切换和自定义主题
color-picker color-picker.js 颜色选择器模块，支持多种颜色格式
ui ui.js UI框架模块，提供UI组件管理
md md.js Markdown编辑器模块

5. 示例代码

以下是一个完整的示例，展示如何使用 Casting UI 框架：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Casting UI 示例</title>
    <link rel="stylesheet" href="dist/v0.2.0/Casting.css">
</head>
<body>
    <div class="container-fixed-md">
        <h1>Welcome to Casting UI</h1>
        
        <!-- 按钮示例 -->
        <button class="btn btn-primary" onclick="showModalExample()">显示弹窗</button>
        <button class="btn btn-secondary" onclick="showToastExample()">显示Toast</button>
        
        <div class="spacer-md"></div>
        
        <!-- 图片示例 -->
        <img src="https://picsum.photos/400/300" class="image" onclick="zoomImage(this.src)">
    </div>
    
    <script src="dist/v0.2.0/Casting.js"></script>
    <script>
        // 初始化框架
        document.addEventListener('DOMContentLoaded', async function() {
            // 初始化主题管理器
            const themeManager = new ThemeManager();
            await themeManager.init();
            
            // 应用默认主题
            themeManager.applyTheme(themeManager.getThemes()[0]);
        });
        
        // 显示弹窗示例
        function showModalExample() {
            showModal({
                title: '示例弹窗',
                content: '这是一个使用 Casting UI 框架创建的弹窗',
                buttons: [
                    {
                        text: '确定',
                        style: 'primary',
                        action: () => {
                            showToast('您点击了确定', 'success');
                        }
                    },
                    {
                        text: '取消',
                        action: () => {
                            showToast('您点击了取消', 'info');
                        }
                    }
                ]
            });
        }
        
        // 显示Toast示例
        function showToastExample() {
            showToast('这是一个 Toast 消息', 'info');
        }
    </script>
</body>
</html>
