核心JS功能

核心JS功能提供框架的基础能力，包括调试、遮罩层和弹窗基类等。

1. 调试功能

调试功能用于在开发环境中输出调试信息。

1.1 debug函数

debug() 函数用于输出调试信息，仅在开发模式下生效。

提示：debug 函数只在 localStorage.getItem('casting_debug') 为 'true' 时才会输出信息到控制台。

参数说明

参数 类型 必填 描述
message String 是 调试信息内容
data Any 否 附加数据，会一并输出

使用说明

// 启用调试模式
localStorage.setItem('casting_debug', 'true');

// 输出调试信息
debug('初始化完成');
debug('用户数据', { name: '张三', age: 25 });

是否涉及 JS 动态渲染：是，纯 JavaScript 功能。

2. 遮罩层组件

遮罩层组件用于创建全屏遮罩，常用于弹窗、加载等场景。

2.1 创建遮罩

createOverlay() 函数用于创建遮罩层元素。

参数说明

函数 返回值 描述
createOverlay() HTMLElement 创建并返回遮罩层元素
showOverlay() HTMLElement 显示遮罩层，返回遮罩元素
hideOverlay() void 隐藏遮罩层

使用说明

// 显示遮罩层
const overlay = showOverlay();

// 在遮罩上显示内容
const modal = document.createElement('div');
modal.innerHTML = '
弹窗内容
';
overlay.appendChild(modal);

// 隐藏遮罩层
hideOverlay();

是否涉及 JS 动态渲染：是，纯 JavaScript 功能。

3. PopupBase 弹窗基类

PopupBase 是所有弹窗组件的基类，提供基础的弹窗功能。

3.1 基础用法

参数说明

方法 参数 描述
show() 无 显示弹窗
hide() 无 隐藏弹窗
destroy() 无 销毁弹窗实例

使用说明

// 继承 PopupBase 创建自定义弹窗
class MyPopup extends PopupBase {
    constructor(options) {
        super(options);
        // 自定义初始化
    }
    
    show() {
        super.show();
        // 自定义显示逻辑
    }
}

// 使用自定义弹窗
const popup = new MyPopup({
    content: '自定义内容'
});
popup.show();

是否涉及 JS 动态渲染：是，纯 JavaScript 功能。

4. 图片放大功能

图片放大功能用于点击图片后全屏查看。

4.1 zoomImage 函数

参数说明

参数 类型 必填 描述
src String 是 图片地址

使用说明

<!-- HTML -->
<img src="image.jpg" class="image" onclick="zoomImage(this.src)">
// JavaScript
zoomImage('https://example.com/image.jpg');

// 关闭图片放大
closeImageZoom();

是否涉及 JS 动态渲染：是，纯 JavaScript 功能。

5. 初始化函数

UI 模块提供多种初始化函数，用于自动绑定组件行为。

5.1 初始化函数列表

函数 描述 自动调用
initCollapsePanels() 初始化折叠面板 是
initModals() 初始化弹窗 是
initToasts() 初始化 Toast 是
initImageZoom() 初始化图片放大 是
initFormValidation() 初始化表单验证 是
initMenus() 初始化菜单 是

使用说明

// 创建 UI 实例时会自动初始化所有组件
const ui = new UI();

// 手动初始化特定组件
ui.initMenus();
ui.initImageZoom();

是否涉及 JS 动态渲染：是，纯 JavaScript 功能。

6. 使用建议

最佳实践：
  • 使用 debug 函数替代 console.log，便于生产环境关闭调试
  • 遮罩层会自动管理 z-index，无需手动设置
  • 继承 PopupBase 时记得调用 super 方法
  • 图片放大功能会自动绑定到带有 .image 类的元素
