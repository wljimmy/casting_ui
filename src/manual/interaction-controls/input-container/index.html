表单控件

表单控件组件提供完整的表单元素，包括输入框、文本域、单选、复选、开关、滑块等。

1. 输入框

基础输入框组件，支持多种类型和状态。

这是提示信息
这是错误信息

参数说明

类名 描述
.input 基础输入框样式
.input.error 错误状态
.input.success 成功状态
.input-group 输入框组容器
.input-label 输入框标签
.input-hint 提示信息
.input-error 错误信息

使用说明

<div class="input-group">
    <label class="input-label">用户名</label>
    <input type="text" class="input" placeholder="请输入用户名">
    <span class="input-hint">用户名长度为3-20个字符</span>
</div>

是否涉及 JS 动态渲染：否，纯 CSS 组件。

2. 文本域

多行文本输入组件。

参数说明

类名 描述
.textarea 文本域样式，支持 resize: vertical

使用说明

<textarea class="textarea" placeholder="请输入内容" rows="4"></textarea>

是否涉及 JS 动态渲染：否。

3. 单选框

单选框组件用于在多个选项中选择一项。

参数说明

类名 描述
.radio-group 单选框组容器
.radio-item 单选框项
.radio-label 单选框标签

使用说明

<div class="radio-group">
    <label class="radio-item">
        <input type="radio" name="gender" value="male">
        <span class="radio-label">男</span>
    </label>
    <label class="radio-item">
        <input type="radio" name="gender" value="female">
        <span class="radio-label">女</span>
    </label>
</div>

是否涉及 JS 动态渲染：否。

5. 开关

开关组件用于表示两种状态之间的切换。

参数说明

类名 描述
.switch 开关容器
.switch-slider 开关滑块

使用说明

<label class="switch">
    <input type="checkbox" checked>
    <span class="switch-slider"></span>
</label>

是否涉及 JS 动态渲染：否。

6. 滑块

滑块组件用于在范围内选择数值。

参数说明

类名 描述
.slider 滑块样式

使用说明

<input type="range" class="slider" min="0" max="100" value="50">

<!-- 带步长 -->
<input type="range" class="slider" min="0" max="100" step="10" value="30">

是否涉及 JS 动态渲染：否。

7. 表单验证

框架提供基础的表单验证样式。

验证状态：
  • .error - 错误状态，边框变红
  • .success - 成功状态，边框变绿
  • .warning - 警告状态，边框变黄

使用说明

// 表单验证示例
const form = document.querySelector('form');
const input = form.querySelector('.input');

// 验证失败
input.classList.add('error');

// 验证成功
input.classList.add('success');

是否涉及 JS 动态渲染：是，需要 JavaScript 进行验证逻辑。

8. 使用建议

最佳实践：
  • 始终为输入框添加 label 标签，提升可访问性
  • 使用 input-hint 提供输入提示，使用 input-error 显示错误信息
  • 单选框和复选框应使用相同的 name 属性进行分组
  • 开关适合表示即时生效的设置项
  • 滑块适合选择范围明确的数值
