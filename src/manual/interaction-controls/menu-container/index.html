使用说明

基本结构

<menu class="menu-sidebar">
    <ul>
        <li data-id="home">首页</li>
        <li data-id="products">产品
            <ul>
                <li data-id="phone">手机</li>
                <li data-id="laptop" data-badge="hot">电脑</li>
            </ul>
        </li>
    </ul>
</menu>

菜单类型

  • menu-sidebar - 侧边栏菜单
  • menu-inline - 内联菜单
  • menu-popup - 弹出式菜单

Data属性

  • data-id - 菜单项唯一标识
  • data-icon - 菜单项图标（SVG路径）
  • data-badge - 菜单项徽章（hot/new）
  • data-action - 点击执行的函数名
  • data-trigger - 弹出菜单触发按钮ID
