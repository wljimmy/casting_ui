菜单API

初始化菜单

菜单组件会在页面加载时自动初始化，无需手动调用。同时会实时监听DOM变化，自动初始化新添加的菜单。

菜单方法

  • ui.menu.init() - 手动初始化菜单
  • ui.menu.toggle(menuId) - 切换菜单的显示/隐藏状态
  • ui.menu.setActive(menuItem) - 设置菜单项为激活状态
  • ui.menu.addItem(menuId, parentItem, itemData) - 动态添加菜单项

addItem方法参数

  • menuId - 菜单ID
  • parentItem - 父菜单项的data-id值，为null时添加到根菜单
  • itemData - 菜单项数据，包含以下属性：
    • id - 菜单项ID（可选）
    • text - 菜单项文本
    • href - 菜单项链接（可选）
    • icon - 菜单项图标（可选）
    • badge - 菜单项徽章（可选），包含text和type属性
    • data - 菜单项数据属性（可选），包含data-*属性

菜单结构

<menu id="menu-id">
    <ul>
        <li data-id="item1">
            <a href="#">
                <div class="menu-content">菜单项1</div>
            </a>
        </li>
        <li data-id="item2">
            <a href="#">
                <div class="menu-content">菜单项2</div>
                <div class="menu-right">
                    <span class="menu-icon">
                        <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
                            <use href="path/to/icon.svg#icon"></use>
                        </svg>
                    </span>
                </div>
            </a>
            <ul>
                <li data-id="subitem1">
                    <a href="#">
                        <div class="menu-content">子菜单项1</div>
                    </a>
                </li>
            </ul>
        </li>
    </ul>
</menu>

Data属性

  • data-id - 菜单项唯一标识
  • data-href - 菜单项跳转链接
  • data-action - 菜单项点击执行的JavaScript代码

特点

  • 支持多级嵌套菜单
  • 自动动画效果
  • 同一级菜单只能展开一个
  • 支持动态添加菜单项
  • 实时监听DOM变化，自动初始化新菜单
  • 支持多个菜单组独立工作
