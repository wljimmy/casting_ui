Input 验证功能测试

CUI-input 验证

输入错误值后失焦（blur），普通输入框显示红边框 + title 提示

CUI-input-box 验证

输入错误值后失焦（blur），组合框显示错误信息

自定义正则验证

使用 data-validate="custom" + data-validate-pattern 自定义正则

CUI-input--simple 验证

简约样式下验证：失焦变红，鼠标悬停显示 title 错误

表单验证

提交前检查整个表单是否有错误项

取消验证的场景

不设 data-validate 的输入框不受验证影响

空值不报错

验证规则：空值通过验证，只有填了内容但格式不对才报错

自动化测试

一键自动填入各种值并验证校验逻辑、属性保留、边界情况
