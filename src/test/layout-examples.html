布局说明

本示例展示了 Casting UI 的两种布局模式：

  • 网页布局：流式自适应，超宽屏时主内容区限宽居中
  • 应用壳布局：上下左右固定，中间主内容区独立滚动

您可以通过上方的控制按钮：

  • 切换布局模式
  • 显示/隐藏各个布局元素
