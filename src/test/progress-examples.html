进度条示例

测试新的进度条调用方式：div#id.CUI-progress + 附加属性

基础进度条（默认：标签+条纹+动画）

不设置data-type时，默认启用标签、条纹和动画效果

<div id="progress1" class="CUI-progress" data-value="60"></div>

简洁模式（plain）

使用 plain 参数关闭所有默认装饰效果

<div id="progress2" class="CUI-progress" data-value="60" data-type="plain"></div>

单独控制各项效果

可以单独启用或关闭某项效果

<div id="progress3" class="CUI-progress" data-value="75" data-type="label"></div>
<div id="progress4" class="CUI-progress" data-value="50" data-type="striped"></div>
<div id="progress5" class="CUI-progress" data-value="50" data-type="striped animated"></div>

毛玻璃效果进度条

<div id="progress6" class="CUI-progress" data-value="80" data-type="glass"></div>

组合效果（标签+条纹）

<div id="progress7" class="CUI-progress" data-value="90" data-type="label striped"></div>

彩色进度条

<div id="progress-success" class="CUI-progress" data-value="85" data-type="success label"></div>
<div id="progress-warning" class="CUI-progress" data-value="65" data-type="warning label"></div>
<div id="progress-error" class="CUI-progress" data-value="35" data-type="error label"></div>
<div id="progress-info" class="CUI-progress" data-value="55" data-type="info label"></div>

自定义颜色进度条

<div id="progress-custom1" class="CUI-progress" data-value="70" data-type="label" data-color="#ff6b6b"></div>
<div id="progress-custom2" class="CUI-progress" data-value="60" data-type="label" data-color="#4ecdc4"></div>
<div id="progress-custom3" class="CUI-progress" data-value="80" data-type="label" data-color="#45b7d1"></div>

自定义尺寸进度条

<div id="progress-size1" class="CUI-progress" data-value="75" data-type="label W300 H20"></div>
<div id="progress-size2" class="CUI-progress" data-value="60" data-type="label W200 H28"></div>
<div id="progress-size3" class="CUI-progress" data-value="85" data-type="label W400 H14"></div>

动态控制（data属性驱动）

进度条默认启用标签、条纹和动画效果，无需额外设置

<div id="progress8" class="CUI-progress" data-value="0"></div>

通过修改 data-value 属性控制进度条：

// 获取进度条值
const value = document.getElementById('progress8').dataset.value;

// 设置进度条值（自动更新UI）
document.getElementById('progress8').dataset.value = 75;
